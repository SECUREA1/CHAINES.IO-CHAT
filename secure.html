Clubhouse Sentinel: Camera Security Tracking Overview

Anchored in the header, this overview mirrors the Clubhouse Sentinel vibe with multi-camera overlays, guest recognition queues, object memory, and automation-ready tracking logs. It runs locally via getUserMedia() and keeps data in localStorage for fast, on-device response cycles that align with clubhouse security routines.

Usage notes
Clubhouse Sentinel is designed for on-device workflows. Run on https (or localhost) for Live CV, and expect a short warm-up on first model load for real-time club floor monitoring.
